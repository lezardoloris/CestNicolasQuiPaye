## Méthodologie

1. **Aucune source partisane n'est utilisée comme donnée primaire.** Les think tanks (IFRAP, FIPECO, OFCE) sont cités en complément, toujours croisés avec des sources institutionnelles.
2. **Toutes les données chiffrées sont traçables** : chaque valeur dans `budget-2026.ts` inclut un champ `source` renvoyant à une institution identifiable.
3. **Les PDFs sont archivés localement** dans ce dossier pour garantir la pérennité en cas de retrait des URLs.
4. **Mise à jour** : ce fichier est mis à jour à chaque ajout de données dans la plateforme.
